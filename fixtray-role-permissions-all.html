FixTray — Role Permissions

All 5 roles & what they can do  ·  Generated May 19, 2026

All Roles
🛡️
Super Admin
Full platform access — overrides all role checks across every tenant
Tenant & Shop Management
View & manage all shops/tenants
Approve or reject shop registrations
Suspend or deactivate any shop
Access any shop's data directly
User Management
View all platform users
Filter users by role
Manage any user account
Platform Operations
View platform-wide analytics
Monitor infrastructure & system health
View audit logs & security settings
Manage deployments
System settings & maintenance mode
View all work orders across all tenants
Bypass subscription gates for any shop
⚠️ Super Admin has unrestricted access to every route and resource on the platform. Use with care.
🏪
Shop Owner
Full control over their shop — operations, team, billing, and settings
Work Orders
Create work orders
View all work orders
Edit work orders
Delete work orders
Assign to technicians
Create work order templates
Team Management
Invite managers & technicians
Remove team members
Set custom per-user permissions
Manage shift schedules
Approve shift swaps & leave requests
Estimates & Inventory
Create estimates
Approve estimates
View inventory
Edit inventory
Finance & Payroll
View & manage employee payroll
View full reports & analytics
Export report data
Manage tax settings
Manage vendors
Timeclock
View all employee time tracking
Manage timeclock entries
Business Settings
Set shop hours & blocked dates
Set appointment capacity
Manage labor rates & services
Configure webhook endpoints
Manage API keys
Upgrade/downgrade subscription plan
Access billing portal
👔
Manager
Day-to-day operations — permissions customizable by Shop Owner
Work Orders
Create work orders
View all shop work orders
Edit work orders
Delete work orders
Assign work orders to technicians
Estimates & Inventory
Create estimates
Approve estimates
View inventory
Edit inventory
Team
View technicians
Manage technicians
View team via shop admin panel
Finance, Reports & Timeclock
View & manage employee payroll
Full access to analytics & reports
View & manage employee time tracking
Settings & Security
View and edit shop settings
Enable/disable two-factor authentication
⚠️ Shop Owner can restrict or expand any of these permissions per individual manager. Managers cannot manage subscriptions or override permissions.
🔧
Technician
Job execution and field tools — permissions customizable by Shop Owner
Work Orders
View assigned work orders
Edit assigned work orders
Mark jobs as complete
Estimates & Inventory
Create estimates
View inventory (read-only)
Inspections & Diagnostics
Create DVI (Digital Vehicle Inspection) reports
Perform and view diagnostics
Upload and manage work photos
Timeclock & Timesheets
Clock in/out
View own timesheet
Submit time entries
Request shifts and time off
Customer & Vehicle
Access customer vehicle history
Send messages to customers & manager
Tools & Field
Access technical manuals & documentation
Access all tech tools
Share real-time location
Create and manage roadside/emergency jobs
⚠️ Techs cannot create/delete work orders, approve estimates, view payroll/reports, or manage other techs by default. Shop Owner can grant additional permissions individually.
🚗
Customer
Self-service portal — view own jobs, pay invoices, manage vehicles
Work Orders
View own work orders only
View estimates for own work orders
Respond to recurring work order requests
Payments
Pay invoices online
Manage saved payment methods
Vehicles & Profile
Add and manage vehicle profiles
Upload and manage documents
Update personal profile information
Communication
Send and receive messages with the shop
Receive work order status notifications
Submit reviews for shops
⚠️ Customers can only see their own data. Accessing another customer's work order returns a 403 Forbidden error.
