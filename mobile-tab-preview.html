How it works
📌  Context-aware: each page shows only the tabs that make sense for that page
🎯  Collapsible: tap the arrow to collapse down to a single thin bar — more screen space for content
🎨  Role-colored: active tab matches the role's accent color (amber=shop, green=tech, cyan=customer, etc.)
💾  Remembers state: collapsed/expanded preference saved per session
📱  Only on sub-pages: home tile grid never shows the tab bar
