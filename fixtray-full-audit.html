Known Limitations & Notes

Fast Refresh Full Reloads (Development Only): When the Next.js dev server performs a full reload after certain file changes, localStorage is cleared, logging out the active session. This is a dev-only behavior — production builds are not affected.
DVI Page (/tech/dvi/[token]): Requires an active work order with a valid DVI token URL. There is no direct navigation to it — a customer or shop sends a specific link. This is by design. No bug.
