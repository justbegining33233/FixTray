FixTray — Role Permissions

What each role can do inside the app  ·  Generated May 19, 2026

v1.0
🏪
Shop Owner
Full control over shop operations, team, and billing
Work Orders
Create work orders
View all work orders
Edit work orders
Delete work orders
Assign to technicians
Create work order templates
Team Management
Invite managers & technicians
Remove team members
Set custom per-user permissions
Manage shift schedules
Approve shift swaps & leave requests
Estimates & Inventory
Create estimates
Approve estimates
View inventory
Edit inventory
Finance & Payroll
View & manage employee payroll
View full reports & analytics
Export report data
Manage tax settings
Manage vendors
Timeclock
View all employee time tracking
Manage timeclock entries
Business Settings
Set shop hours & blocked dates
Set appointment capacity
Manage labor rates & services
Configure webhook endpoints
Manage API keys
Upgrade/downgrade subscription plan
Access billing portal
👔
Manager
Day-to-day operations — can be customized by Shop Owner
Work Orders
Create work orders
View all shop work orders
Edit work orders
Delete work orders
Assign work orders to technicians
Estimates & Inventory
Create estimates
Approve estimates
View inventory
Edit inventory
Team
View technicians
Manage technicians
View team via shop admin panel
Finance, Reports & Timeclock
View & manage employee payroll
Full access to analytics & reports
View & manage employee time tracking
Settings & Security
View and edit shop settings
Enable/disable two-factor authentication
⚠️ Shop Owner can restrict or expand any of these permissions per individual manager. Managers cannot manage subscriptions or override permissions.
🔧
Technician
Job execution and field tools — can be customized by Shop Owner
Work Orders
View assigned work orders
Edit assigned work orders
Mark jobs as complete
Estimates & Inventory
Create estimates
View inventory (read-only)
Inspections & Diagnostics
Create DVI (Digital Vehicle Inspection) reports
Perform and view diagnostics
Upload and manage work photos
Timeclock & Timesheets
Clock in/out
View own timesheet
Submit time entries
Request shifts and time off
Customer & Vehicle
Access customer vehicle history
Send messages to customers & manager
Tools & Field
Access technical manuals & documentation
Access all tech tools
Share real-time location
Create and manage roadside/emergency jobs
⚠️ Techs cannot create/delete work orders, approve estimates, view payroll/reports, or manage other techs by default. Shop Owner can grant additional permissions individually.
👤
Customer
Self-service portal — limited to their own data only
Work Orders
View own work orders only
View estimates on own work orders
Payments
Pay invoices online
Manage saved payment methods
Vehicles & Documents
Manage vehicle profiles
Upload and manage personal documents
Communication
Send and receive messages with shop/tech
Receive work order status notifications
Respond to recurring work order requests
Profile & Reviews
Update personal profile information
Submit reviews for shops
⚠️ Customers are strictly isolated — they cannot view other customers' data, create/edit work orders, access reports, or touch any shop operations. A 403 error is returned if they attempt to access another customer's work order.
⚙️
Superadmin
Platform-level administration — full override on all routes
Tenant & User Management
Manage all tenants (shops) on the platform
View all platform users, filter by role
Bypass subscription gates for all roles
Analytics & Infrastructure
Platform-wide analytics
Infrastructure monitoring
Manage deployments
Security & Audit
View full audit logs
Manage platform security settings
System
System settings and maintenance mode
View all work orders across all tenants
⚠️ Superadmin has full platform override. The legacy "admin" role is automatically promoted to "superadmin" at login. Superadmin is intended for platform operators only, not shop staff.
